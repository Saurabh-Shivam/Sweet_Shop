Problem: "Cannot find module" errors
Solution: Delete node_modules and reinstall
    cd backend
    rm -rf node_modules
    npm install
    OR on Windows: rmdir /s node_modules then npm install

Problem: "MongoDB connection error"
Solution: Ensure MongoDB is running
    Check: brew services list (macOS)
    Check: Services in Windows
    Try: mongod command to start

Problem: "Port 5000 already in use"
Solution: Change backend port
    Edit: backend/.env
    Change: PORT=5000 to PORT=5001
    Update: frontend/.env
    Change: REACT_APP_API_URL=http://localhost:5000/api
      to: REACT_APP_API_URL=http://localhost:5001/api

Problem: CORS errors in browser
Solution: Check backend is running and .env files are correct

Problem: Tests failing
Solution: Ensure MongoDB is running and test database is accessible
